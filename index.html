Inicio

Mi primer acercamiento a la programación

Comenzar a programar fue un desafío interesante. Al principio todo parecía complicado, pero con práctica constante fui entendiendo conceptos clave. Hoy puedo decir que cada error fue una oportunidad para aprender algo nuevo.

  • Aprendí la estructura básica de HTML
  • Entendí la importancia de la lógica
  • Comencé a crear mis primeras páginas web

Errores que me ayudaron a mejorar

Durante el proceso cometí muchos errores, pero algunos fueron fundamentales para crecer. Equivocarse es parte del aprendizaje y permite entender mejor cómo funcionan las cosas.

  • No cerrar correctamente etiquetas
  • Olvidar atributos importantes
  • No organizar bien el código
