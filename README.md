# Blog Personal

## Descripción
Proyecto realizado con HTML5 y CSS3 para practicar estructura semántica, navegación interna y estilos básicos. Simula un blog personal donde se presentan experiencias de aprendizaje en programación, habilidades, intereses y datos de contacto (ficticios).

## Tecnologías utilizadas
- HTML5
- CSS3
